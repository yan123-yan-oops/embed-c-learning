### 🔍 代码阅读题

**题 A：**
```c
struct {
    uint8_t a;
    uint32_t b;
    uint8_t c;
} s;
printf("%zu", sizeof(s));
// 输出？为什么不是 1+4+1=6？
```

**题 B：**
```c
union {
    uint32_t word;
    uint8_t  bytes[4];
} u;
u.word = 0x12345678;
printf("0x%02X", u.bytes[0]);
// 小端系统输出？
```
